Spoofing Your MAC Address

By Ryan Hart January 22, 2015

Changing your MAC address on a linux based machine is fairly straight forward and can be extremely useful.

The way to do the is as follows:

Open a terminal window (my preference is iTerm on my MacBook Pro) Run ~~~ ifconfig ~~~ and you should see a screen similar to the one shown below. (on MacBook Pros en0 is Ethernet, and en1 is Wireless) 1. This is a numbered list. 2. I’m going to include a fenced code block as part of this bullet:

```
Code
More Code
```
  1. Run ifconfig en1 | grep ether`.
  2. You should see ether 00:00:00:00:00:00 with your specific MAC address. The reason en1 is used is because you want to find the MAC address for your wireless card. (To reset your MAC address, you will either want to write it down, or it will automagically reset when you reboot your computer)
  3. To change your MAC address run sudo ifconfig en1 ether 00:00:00:00:00:01 with the new MAC address that you want your wireless card to have.
  4. To check that it worked you can run ifconfig en1 | grep ether and the new MAC address should be printed to the screen.
